Необходимо:
1. Скачать "Платформу для обучения" с сайта http://online.1c.ru/catalog/free/18610119/ по кнопке "Получить продукт бесплатно".
2. Установить "Платформу для обучения".
3. Зайти в 1С:Предприятие, добавить новую базу, запустить конфигуратор
4. Выполнить задание из файла "Задание"
5. Дополнительно прикладываются книги по программированию
	- "Руководство разработчика"
	- "М.Г.Радченко 1C Предприятие 8.2 Практическое пособие разработчика"
