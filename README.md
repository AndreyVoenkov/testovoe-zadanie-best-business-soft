# test-task-best-business-soft

Тестовое задание 1С: учебная конфигурация складского учета (товары, клиенты, документ продажи, регистр Продажи, отчет).

## Условие задания (кратко)

- Справочники: Товары (цена, проверка уникальности наименования), Клиенты.  
- Документ продажи с табличной частью, автоподстановкой цены, пересчетом суммы.  
- Печатная форма накладной с сортировкой строк при выводе (без изменения табличной части).  
- Регистр накопления "Продажи" (товар, клиент, количество, сумма).  
- Отчет по продажам с отбором по периоду, товару, клиенту.

Полный текст задания см. в файле `Задание.txt`.

## Где смотреть код

  `Catalogs/Товары/Ext/ObjectModule.bsl`

  `Documents/РеализацияТоваров/Ext/ObjectModule.bsl`
- Печатная форма и сортировка строк при выводе:  
  `Documents/РеализацияТоваров/Forms/ФормаДокумента/Ext/Module.bsl`
- Отчет по продажам:
