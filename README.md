Учебная конфигурация складского учета на 1С:Enterprise 8.3. Реализованы справочники Товары и Клиенты, документ продажи с движениями по регистру Продажи и печатной формой, а также отчет по продажам с отборами по периоду, товару и клиенту.

# Тестовое задание от Бест Бизнес Софт

Тестовое задание 1С: учебная конфигурация складского учета (товары, клиенты, документ продажи, регистр Продажи, отчет).

## Реализованный функционал и объекты

- Справочники: Товары, Клиенты.  
- Документ продажи: оформление продажи клиенту с табличной частью товаров, автоподстановкой цены и пересчетом суммы строки.  
- Печатная форма накладной: вывод документа с возможностью сортировки строк по выбору пользователя (по длине наименования товара, по наименованию, по цене) без изменения порядка строк в документе.  
- Регистр накопления: Продажи (учет количества и суммы продаж в разрезе товаров и клиентов), движения формируются документом продажи.  
- Отчет по продажам: формируется по регистру Продажи, поддерживает одновременные отборы по периоду, товару и клиенту.  
- Дополнительно: реализована проверка уникальности наименования товара при записи элемента справочника.
